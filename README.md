# CSE425 Project: Unsupervised Music Generation

This repository contains the full project for CSE425, focusing on unsupervised music generation using various models including Autoencoders, VAEs, Transformers, and RLHF.

## Project Structure

- `music-generation-unsupervised/`: Main project directory containing source code, notebooks, and models.
- `rlhf_music.py`: Reinforcement Learning from Human Feedback implementation.
- `evaluate_models.py`: Evaluation scripts for comparing different generative models.
- `generate_report.py`: Script to generate the final project report.
- `outputs/`: Generated MIDI files and model checkpoints (excluded from git by default).

## Getting Started

Refer to the [README](music-generation-unsupervised/README.md) inside `music-generation-unsupervised/` for detailed instructions on setting up the environment and running the models.
